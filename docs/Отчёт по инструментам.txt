
**Отчет: Инструменты для инженера‑конструктора**  
**Дата:** 2026‑02‑08

---

## 1. Общая идея  
Сайт содержит два набора инструментов:  
- **упрощенные** (для быстрых оценок),  
- **СП‑ориентированные** (второй блок).

Каждый инструмент — отдельный расчет. Вводятся исходные данные, инструмент возвращает ключевые величины (момент, прогиб, требуемую арматуру и т.д.).

---

## 2. Упрощенные инструменты (верхний блок)

### 2.1 Подбор балки по моменту + прогиб  
**Схема:** однопролетная балка, равномерно распределенная нагрузка.  
**Формулы:**
- Максимальный момент:  
  **M = w · L² / 8**
- Модуль сопротивления прямоугольного сечения:  
  **W = b · h² / 6**
- Напряжение:  
  **σ = M / W**
- Момент инерции:  
  **I = b · h³ / 12**
- Прогиб:  
  **f = 5 · w · L⁴ / (384 · E · I)**
- Проверка:  
  **σ ≤ fᵧ** и **f ≤ L / (предельное)**

---

### 2.2 Сбор нагрузок  
**Формула (упрощённая):**  
**N = γG·G + γQ₁·Qlong + γQ₂·Qshort**

---

### 2.3 Анкеры  
**Учет:** только сталь болта (без разрушения бетона).  
- Площадь одного болта:  
  **A = π·d² / 4**
- Несущая по растяжению:  
  **Nᵣd = n · A · fᵧ / γM**
- Несущая по срезу:  
  **Vᵣd = n · 0.6 · A · fᵧ / γM**
- Проверка по взаимодействию:  
  **(N/Nᵣd)² + (V/Vᵣd)² ≤ 1**

---

### 2.4 Сварные швы  
**Угловой шов, срез по горловине.**  
- Горловина: **a = 0.7 · s**  
- Несущая:  
  **Vᵣd = a · L · fᵥw / γM**

---

### 2.5 Устойчивость колонны (Эйлер)  
**Критическая сила:**  
**Pcr = π² · E · I / (k·L)²**  
Проверка: **N / Pcr ≤ 1**

---

### 2.6 Армирование плиты (упрощенно)  
**На полосе 1 м:**  
- рычаг внутренних сил **z ≈ 0.9·h₀**  
- требуемая арматура:  
  **As = M / (0.87 · fyd · z)**  
- шаг стержней:  
  **s = Abar · 1000 / As**

---

### 2.7 Свайный фундамент (упрощенно)  
**Сопротивление сваи = боковое + основание:**  
- Боковое:  
  **Qₛ = Σ (fᵢ · α · u · Δz)**  
- Основание:  
  **Qb = qᵦ · A**  
- Полное сопротивление:  
  **Q = Qₛ + Qb**

## 3. Инструменты по СП 63.13330.2018

### 3.1 ЖБ балка (СП 63)  
**Прямоугольное сечение:**  
- Условие равновесия:  
  **M = Rb · b · (h₀·x − 0.5·x²)**  
- По этому уравнению определяется **x** (сжатая зона).  
- Требуемая арматура:  
  **As = (Rb · b · x) / Rs**  
- Контроль:  
  **x ≤ ξR · h₀**

---

### 3.2 ЖБ плита (СП 63)  
**Расчет по полосе 1 м:**  
- Используется то же уравнение, что и для балки:  
  **M = Rb · b · (h₀·x − 0.5·x²)**, где **b = 1000 мм**  
- Арматура:  
  **As = (Rb · b · x) / Rs**  
- Шаг стержней:  
  **s = Abar · 1000 / As**  
- Контроль:  
  **x ≤ ξR · h₀**

---

### 3.3 ЖБ колонна (СП 63)  
**Центральное сжатие (упрощенно):**  
- **N = Rb · Ac + Rs · As**  
где **Ac = b·h − As**

---
