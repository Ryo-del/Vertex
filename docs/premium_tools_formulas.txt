Премиум-инструменты: формулы и логика

1) Пакетный расчет балок (batch/beam)
   Используется тот же расчет, что и в инструменте «Подбор балки по моменту + прогиб».
   Для каждой строки:
   - Максимальный момент: M = w * L^2 / 8
   - Модуль сопротивления прямоугольного сечения: W = b * h^2 / 6
   - Напряжение: σ = M / W
   - Момент инерции: I = b * h^3 / 12
   - Прогиб: f = 5 * w * L^4 / (384 * E * I)
   Проверки: σ <= fy и f <= L/(предельное)

2) Автоподбор балки (auto/beam)
   Вход: материал, пролет, равномерная нагрузка, ширина b.
   Алгоритм:
   - Принимается h = 0 (не задано).
   - Для удовлетворения прочности выбирается высота:
     Wreq = M / fy
     h = sqrt(6 * Wreq / b)
   Далее рассчитываются σ и f по формулам как в (1).

3) Рекомендация сварного шва (recommend/weld)
   Задача: подобрать размер углового шва s по заданной поперечной силе.
   Используется формула несущей способности углового шва:
   - Горловина: a = 0.7 * s
   - Несущая способность: V = (a * L * fvw / γM) / 1000
   Отсюда:
   s = (V * 1000 * γM) / (0.7 * L * fvw)
   Минимальный размер ограничен 3 мм.

4) Импорт из Excel (import/beam)
   Импортирует таблицу для балок и считает по формулам (1).
   Формат строки:
   material, span_m, udl_kn_m, width_m, height_m(optional), fy, e, defl_limit
   Если height_m не задан — алгоритм автоподбора (см. п.2).

Примечание:
- Все расчеты являются инженерными оценками (упрощенные формулы).
- Для нормативного расчета нужно применять конкретные СП/ГОСТ и их коэффициенты.
